Ultimate Image/Video Slider for Madison

About Ultimate I/V Slider

We think mostly for creating sliders at an start. This free extension provides the full featured resource with sample data also, which includes a slider with static banner, iframe video and an .mp4 video (Uploaded to local server). This also provides an direct menu shortcut to frontend

Key Features

* Ultimate I/V Slider > Upload static image banner (.jpg, .jpeg, .png, .gif)
* Ultimate I/V Slider > Upload video from video sharing sites such as youtube, dailymotion, etc., via IFrame.
* Ultimate I/V Slider > Upload video from local source in .mp4 and load via browser default player.
* Frontend > Access frontend directly from admin panel of magento.

Support and Guarantee
